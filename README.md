# RJT NEXUS360

Suite Executiva Inteligente by RJT Consultoria.

SaaS empresarial para consolidar indicadores, importar dados operacionais e gerar dashboards executivos em uma experiencia corporativa clara, limpa e premium.

Cliente piloto: União Bag.

## Rodar Localmente

**Pre-requisito:** Node.js

1. Instale dependencias:
   `npm install`
2. Configure variaveis locais em `.env`.
3. Rode o app:
   `npm run dev`

## Scripts

- `npm run dev`: inicia servidor local.
- `npm run lint`: valida TypeScript.
- `npm run build`: gera build de producao.
